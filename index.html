MoodWave Access Portal

Build Better Spaces with Real Sensor Intelligence

MoodWave is an app that uses an Arduino sensor board to measure room conditions like temperature, humidity, sound, and brightness. The system analyzes these values to make assumptions about the current space and mood.

By using the environmental data, MoodWave creates an environment that increases productivity, reduces stress, and stimulates creativity.
